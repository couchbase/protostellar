Services

service SearchAdminService

SearchAdminService provides search index management and control.

rpc GetIndex
Request: couchbase.admin.search.v1.GetIndexRequest
Response: couchbase.admin.search.v1.GetIndexResponse

GetIndex retrieves a single search index by name.

rpc ListIndexes
Request: couchbase.admin.search.v1.ListIndexesRequest
Response: couchbase.admin.search.v1.ListIndexesResponse

ListIndexes returns all search indexes, optionally scoped.

rpc CreateIndex
Request: couchbase.admin.search.v1.CreateIndexRequest
Response: couchbase.admin.search.v1.CreateIndexResponse

CreateIndex creates a new search index.

rpc UpdateIndex
Request: couchbase.admin.search.v1.UpdateIndexRequest
Response: couchbase.admin.search.v1.UpdateIndexResponse

UpdateIndex updates an existing search index definition.

rpc DeleteIndex
Request: couchbase.admin.search.v1.DeleteIndexRequest
Response: couchbase.admin.search.v1.DeleteIndexResponse

DeleteIndex deletes a search index.

rpc AnalyzeDocument
Request: couchbase.admin.search.v1.AnalyzeDocumentRequest
Response: couchbase.admin.search.v1.AnalyzeDocumentResponse

AnalyzeDocument analyzes a document against an index definition, returning the tokens that would be indexed.

rpc GetIndexedDocumentsCount
Request: couchbase.admin.search.v1.GetIndexedDocumentsCountRequest
Response: couchbase.admin.search.v1.GetIndexedDocumentsCountResponse

GetIndexedDocumentsCount returns the number of documents indexed by a specific search index.

rpc PauseIndexIngest
Request: couchbase.admin.search.v1.PauseIndexIngestRequest
Response: couchbase.admin.search.v1.PauseIndexIngestResponse

PauseIndexIngest pauses document ingestion for an index.

rpc ResumeIndexIngest
Request: couchbase.admin.search.v1.ResumeIndexIngestRequest
Response: couchbase.admin.search.v1.ResumeIndexIngestResponse

ResumeIndexIngest resumes document ingestion for a paused index.

rpc AllowIndexQuerying
Request: couchbase.admin.search.v1.AllowIndexQueryingRequest
Response: couchbase.admin.search.v1.AllowIndexQueryingResponse

AllowIndexQuerying allows queries against an index.

rpc DisallowIndexQuerying
Request: couchbase.admin.search.v1.DisallowIndexQueryingRequest
Response: couchbase.admin.search.v1.DisallowIndexQueryingResponse

DisallowIndexQuerying prevents queries against an index.

rpc FreezeIndexPlan
Request: couchbase.admin.search.v1.FreezeIndexPlanRequest
Response: couchbase.admin.search.v1.FreezeIndexPlanResponse

FreezeIndexPlan prevents the index plan from being reassigned.

rpc UnfreezeIndexPlan
Request: couchbase.admin.search.v1.UnfreezeIndexPlanRequest
Response: couchbase.admin.search.v1.UnfreezeIndexPlanResponse

UnfreezeIndexPlan allows the index plan to be reassigned.

message AnalyzeDocumentRequest

AnalyzeDocumentRequest analyzes a document against an index definition.

Field Type Description
1 name string

The index name to analyze against.
2 doc bytes

The document to analyze. JSON-encoded.
3 bucket_name optional string
4 scope_name optional string

message AnalyzeDocumentResponse

AnalyzeDocumentResponse contains the analysis results.

Field Type Description
1 status string

Status of the analysis operation.
2 analyzed bytes

The analysis results (tokens, etc.). JSON-encoded.

message Index

Index represents a full-text search index definition.

Field Type Description
1 name string

The name of the index.
2 params map<string, bytes>

Index parameters (type mappings, analyzers, etc.). Values are JSON-encoded.
3 plan_params map<string, bytes>

Plan parameters (num_replicas, partition settings). Values are JSON-encoded.
4 source_name optional string

The data source name (typically the bucket name).
5 source_params map<string, bytes>

Source-specific parameters. Values are JSON-encoded.
6 source_type optional string

The type of data source (e.g., "couchbase").
7 source_uuid optional string

The UUID of the source.
8 type string

The index type (e.g., "fulltext-index").
9 uuid string

The unique identifier for this index.
