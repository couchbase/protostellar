Services

service SearchService

SearchService provides full-text search query execution.

rpc SearchQuery
Request: couchbase.search.v1.SearchQueryRequest
Response: couchbase.search.v1.SearchQueryResponse

SearchQuery executes a search query against a named index and streams back hits, facet results, and metadata.

enum HighlightStyle

HighlightStyle controls how matching terms are highlighted in results.

Name Number Description
HIGHLIGHT_STYLE_DEFAULT 0

Server default highlighting.
HIGHLIGHT_STYLE_HTML 1

HTML tag highlighting ().
HIGHLIGHT_STYLE_ANSI 2

ANSI escape code highlighting (for terminals).

enum KnnOperator

KnnOperator controls how multiple KNN queries are combined.

Name Number Description
KNN_OPERATOR_OR 0

Results from multiple KNN queries are combined with OR (union).
KNN_OPERATOR_AND 1

Results from multiple KNN queries are combined with AND (intersection).

enum Operator

Operator controls how multiple terms in the value are combined.

Name Number Description
OPERATOR_OR 0

Any term may match (default).
OPERATOR_AND 1

All terms must match.
