Services

service QueryAdminService

QueryAdminService provides N1QL index lifecycle management.

rpc GetAllIndexes
Request: couchbase.admin.query.v1.GetAllIndexesRequest
Response: couchbase.admin.query.v1.GetAllIndexesResponse

GetAllIndexes returns all indexes matching the optional scope filter.

rpc CreatePrimaryIndex
Request: couchbase.admin.query.v1.CreatePrimaryIndexRequest
Response: couchbase.admin.query.v1.CreatePrimaryIndexResponse

CreatePrimaryIndex creates a primary index on a collection. A primary index indexes all document keys.

rpc CreateIndex
Request: couchbase.admin.query.v1.CreateIndexRequest
Response: couchbase.admin.query.v1.CreateIndexResponse

CreateIndex creates a secondary index on specific document fields.

rpc DropPrimaryIndex
Request: couchbase.admin.query.v1.DropPrimaryIndexRequest
Response: couchbase.admin.query.v1.DropPrimaryIndexResponse

DropPrimaryIndex removes a primary index.

rpc DropIndex
Request: couchbase.admin.query.v1.DropIndexRequest
Response: couchbase.admin.query.v1.DropIndexResponse

DropIndex removes a named secondary index.

rpc BuildDeferredIndexes
Request: couchbase.admin.query.v1.BuildDeferredIndexesRequest
Response: couchbase.admin.query.v1.BuildDeferredIndexesResponse

BuildDeferredIndexes triggers the build of all indexes that were created with deferred=true. Returns the list of indexes that will be built.

rpc WaitForIndexOnline
Request: couchbase.admin.query.v1.WaitForIndexOnlineRequest
Response: couchbase.admin.query.v1.WaitForIndexOnlineResponse

WaitForIndexOnline blocks until the specified index reaches ONLINE state.

Enums

enum IndexState

IndexState represents the current lifecycle state of an index.

Name Number Description
INDEX_STATE_DEFERRED 0

The index was created with deferred=true and has not been built yet.
INDEX_STATE_BUILDING 1

The index is currently being built.
INDEX_STATE_PENDING 2

The index build is pending (queued).
INDEX_STATE_ONLINE 3

The index is online and ready for queries.
INDEX_STATE_OFFLINE 4

The index is offline.
INDEX_STATE_ABRIDGED 5

The index is in an abridged state.
INDEX_STATE_SCHEDULED 6

The index build is scheduled but not yet started.

enum IndexType

IndexType specifies the underlying index technology.

Name Number Description
INDEX_TYPE_VIEW 0

A MapReduce view-based index.
INDEX_TYPE_GSI 1

A Global Secondary Index (GSI).
