Services

service CollectionAdminService

CollectionAdminService provides CRUD for scopes and collections.

rpc ListCollections
Request: couchbase.admin.collection.v1.ListCollectionsRequest
Response: couchbase.admin.collection.v1.ListCollectionsResponse

ListCollections returns all scopes and collections within a bucket, along with the current manifest_uid.

rpc CreateScope
Request: couchbase.admin.collection.v1.CreateScopeRequest
Response: couchbase.admin.collection.v1.CreateScopeResponse

CreateScope creates a new scope within a bucket.

rpc DeleteScope
Request: couchbase.admin.collection.v1.DeleteScopeRequest
Response: couchbase.admin.collection.v1.DeleteScopeResponse

DeleteScope removes a scope and all its collections from a bucket.

rpc CreateCollection
Request: couchbase.admin.collection.v1.CreateCollectionRequest
Response: couchbase.admin.collection.v1.CreateCollectionResponse

CreateCollection creates a new collection within a scope.

rpc UpdateCollection
Request: couchbase.admin.collection.v1.UpdateCollectionRequest
Response: couchbase.admin.collection.v1.UpdateCollectionResponse

UpdateCollection modifies a collection's configuration (e.g., max expiry).

rpc DeleteCollection
Request: couchbase.admin.collection.v1.DeleteCollectionRequest
Response: couchbase.admin.collection.v1.DeleteCollectionResponse

DeleteCollection removes a collection from a scope.
